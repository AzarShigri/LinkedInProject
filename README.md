# LinkedIn UI Clone - React Native

A simple and modern **LinkedIn-inspired mobile UI** built using **React Native**.  
This project demonstrates the use of components, hooks, state management, and responsive styling in React Native.

---

## 📱 Features

- Beautiful LinkedIn-style UI
- Profile Card Section
- Dynamic Feed Posts
- Like / Unlike Functionality ❤️
- Responsive Design
- React Hooks (`useState`, `useEffect`)
- Clean and reusable code structure

---

## 🚀 Technologies Used

- React Native
- JavaScript (ES6)
- React Hooks
- StyleSheet API
